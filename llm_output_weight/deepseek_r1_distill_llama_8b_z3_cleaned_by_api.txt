=== Cleaning Problem 27 ===


```python
from z3 import *

# Define variables
air_per_tire = Real('air_per_tire')
total_air = 4 * air_per_tire

# Total weight without air: car + 4 tires
weight_without_air = 820 + 4 * 9.8

# Total weight with air should equal scale reading
s = Solver()
s.add(weight_without_air + total_air == 859.2)

# Solve
if s.check() == sat:
    m = s.model()
    result = m[air_per_tire]
    print(result)
else:
    print("No solution found")
```

    m = s.model()
